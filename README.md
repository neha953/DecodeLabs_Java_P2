# DecodeLabs Java Project 2 - Student Grade Calculator

## Overview
A console-based Student Grade Calculator built in Java as part of the DecodeLabs Industrial Training Kit - Batch 2026.

## How It Works
The program takes a student name and marks for multiple subjects out of 100. It calculates total marks and average percentage using type casting for decimal precision, assigns a grade based on a conditional logic ladder, and displays a clean formatted result card.

## Features
- Accepts marks for any number of subjects
- Input validation - rejects marks below 0 or above 100
- Total marks calculation using accumulator loop
- Average percentage with decimal precision using type casting
- Grade assignment using Logic Ladder
  - A+ for 90 and above
  - A for 80 and above
  - B for 70 and above
  - C for 60 and above
  - D for 50 and above
  - F below 50
- Pass or Fail status
- Clean formatted result card using printf
- Scanner buffer trap fix using Integer.parseInt

## Concepts Covered
- Variables and data types
- For loops and while loops
- Arithmetic operations
- Type casting
- Conditional statements
- Input validation
- Formatted output with printf
